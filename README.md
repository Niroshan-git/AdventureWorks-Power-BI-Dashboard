# AdventureWorks Power BI Dashboard

## Overview
This Power BI dashboard was created to provide an insightful analysis of sales and performance metrics for AdventureWorks. The report is structured into several sections to enable deep-dive analytics and actionable insights for stakeholders.

## Executive Dashboard

![Main Dashboard](https://github.com/user-attachments/assets/b8d75c25-e158-4974-9b81-91046074b603)

The **Executive Dashboard** provides a high-level overview of key performance indicators (KPIs) such as:
- **Total Revenue**: Displays cumulative revenue generated.
- **Profit**: Tracks net profit.
- **Orders**: Shows total orders processed.
- **Return Rate**: Calculates the percentage of returns.

### Visual Highlights:
- **Revenue Trending**: A line graph displaying revenue trends over time.
- **Orders by Category**: A breakdown of orders by product categories (e.g., Accessories, Bikes, Clothing).
- **Top 10 Products**: Lists top-performing products by orders, revenue, and return percentage.
  
## Map Visualization

![Map Visual](https://github.com/user-attachments/assets/01431a4e-b072-48e4-9adf-e23e7a37f8e5)

The **Map** page visualizes sales distribution by country, with regional filters (e.g., North America, Europe, Pacific) for focused analysis. Larger markers indicate higher sales volumes, helping to identify top-performing locations.

## Product Details

![product detail](https://github.com/user-attachments/assets/dd766444-198c-4cff-8a76-d9bfa7277312)

The **Product Details** section offers an in-depth view of selected products with metrics including:
- **Monthly Orders vs. Target**
- **Monthly Revenue vs. Target**
- **Monthly Profit vs. Target**
- **Return % Trend**: Shows return trends to identify areas for improvement.

This page also includes interactive elements, allowing users to adjust price and analyze profit impacts dynamically.

## Customer Details

![Customer Pane](https://github.com/user-attachments/assets/3d42d656-1542-4375-8343-27288aa97536)

The **Customer Details** page (not pictured in the screenshots) provides insights into customer demographics and purchasing behavior, supporting targeted marketing and customer retention strategies.

## Getting Started
To use or modify this report:
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/adventureworks-dashboard.git  ```

- **Open in Power BI Desktop**: Open the `.pbix` file in Power BI Desktop to view or modify the dashboard.
- **Dataset**: This report uses AdventureWorks sample data. Ensure your data model is correctly linked if using alternative datasets.

### Technical Details
- **Tools Used**: Power BI Desktop, AdventureWorks sample dataset
- **Visualizations**: Line charts, bar charts, donut charts, and map visuals.
- **Interactivity**: Filters, slicers, and interactive drill-downs.
